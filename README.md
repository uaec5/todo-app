## Getting Started

1. Clone the repository:
```bash
git clone [your-repo-url]
cd todo-app
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
ng serve
```

4. Open your browser and navigate to `http://localhost:4200`
